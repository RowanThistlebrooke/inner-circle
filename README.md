# The Inner Circle

One routine per episode. Each folder is a drop: the one prompt that does
everything the video showed, plus the judgement rules the free version
teaches by hand.

- `finance-stocks/` — the finance routine: setup, the daily refresh, THE
  FILTER LAW, the pick and health rules.

Members only. The free half of every drop lives in the public episode repos.

Naming note: "vault" always means the USER'S vault - their ledger, equation
and context on their own board. This repo is the Inner Circle: the routines.
